# cAgents-Creative

**Creative Writing Domain Plugin**

Coming in Phase 2 - 18 specialized agents for novels, stories, and worldbuilding.

## Planned Agents

### Narrative and Structure
- Story Architect - Overall vision, structure, creative direction
- Plot/Structure Designer - Story structure, pacing, arcs
- Scene Crafter - Individual scene construction

### Worldbuilding
- Character Developer - Character profiles, arcs, psychology
- Location/Geography Specialist - Maps, places, settings
- Systems Designer - Magic systems, technology, rules
- Historian/Timeline Keeper - Historical events, chronology

### Writing Craft
- Writer - Primary prose generation
- Dialogue Specialist - Character voice, conversation
- Style/Voice Editor - Narrative voice, tone consistency

### Quality and Coherence
- Continuity Checker - Cross-element consistency
- Pacing Analyst - Rhythm, tension, release
- Emotion/Theme Tracker - Emotional arcs, themes

### Workflow
- Router - Creative-specific complexity classification
- Planner - Creative project decomposition
- Executor - Orchestrates creative team
- Validator - Continuity, style, structural completeness
- Self-Correct - Creative-specific recovery

## Installation

```bash
# Coming in Phase 2
claude /plugin install @cagents/creative
```

## Usage

```bash
/trigger Write a novel about space pirates
/trigger Create a character profile for a reluctant hero
/trigger Design a magic system based on emotions
```
