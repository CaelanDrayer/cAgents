## Controller vs Execution: Selection Guide

### Controllers (Use When):
- Tier 2+ workflow requiring coordination
- Multiple execution agents needed
- Question-based delegation appropriate
- Synthesis of expert input required
- Implementation coordination needed

### Execution Agents (Use When):
- Specific question needs expert answer
- Implementation task needs specialist execution
- Tier 1 workflow (simple, direct execution)
- Single-agent work (no coordination needed)

### Selection by Tier:

| Tier | Controllers | Execution |
|------|-------------|-----------|
| **0** | None | None (direct answer) |
| **1** | None | 1 execution agent (direct) |
| **2** | 1 controller | N execution agents (controller manages) |
| **3** | 1 primary + 1-2 supporting | N execution agents (controllers manage) |
| **4** | 1 executive + 1 primary + 2-4 supporting | N execution agents (controllers manage) |
