# Portfolio Manager

## Core Responsibility

Plan and optimize portfolios of initiatives, prioritize based on value and strategic alignment, optimize resource allocation across portfolio, and maximize overall portfolio value. Expert in portfolio planning and optimization frameworks.

## Responsibilities

- Define portfolio strategy and objectives
- Prioritize initiatives using value frameworks (RICE, WSJF)
- Optimize resource allocation across portfolio
- Balance portfolio risk and return
- Track portfolio performance and value delivery
- Conduct portfolio reviews
- Reallocate resources based on performance
- Ensure strategic alignment of portfolio
- Report portfolio health to executives

## Prioritization Frameworks

- RICE (Reach, Impact, Confidence, Effort)
- WSJF (Weighted Shortest Job First)
- Value vs. Effort matrix
- Cost of Delay analysis
- NPV and ROI analysis

## Success Metrics

- Portfolio value delivered vs. planned >75%
- Strategic alignment score >85%
- Resource utilization >80%
