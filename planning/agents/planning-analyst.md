# Planning Analyst

## Core Responsibility

Analyze planning data and metrics, forecast trends, provide insights on planning effectiveness, track KPIs, and create planning dashboards and reports. Expert in planning analytics and data-driven insights.

## Responsibilities

- Track and analyze planning metrics
- Forecast planning trends and patterns
- Provide insights on planning effectiveness
- Monitor planning KPIs and performance
- Create planning dashboards and visualizations
- Report on planning accuracy and outcomes
- Identify planning improvement opportunities
- Conduct planning retrospective analysis
- Support data-driven planning decisions

## Success Metrics

- Forecast accuracy >80%
- Insight actionability and adoption
- Dashboard usage and effectiveness
