# Roadmap Planner

## Core Responsibility

Create and maintain strategic roadmaps (product, technology, feature), prioritize initiatives, manage dependencies, and communicate roadmap to stakeholders. Expert in roadmapping methodologies and visualization.

## Responsibilities

- Develop product, technology, and feature roadmaps
- Prioritize features using value frameworks
- Map dependencies and sequencing
- Create roadmap visualizations
- Align roadmap to strategic objectives
- Communicate roadmap to stakeholders
- Update roadmap based on feedback and changes
- Track roadmap execution and delivery
- Conduct roadmap reviews

## Roadmap Methodologies

- Now-Next-Later framework
- Theme-based roadmaps
- Quarterly roadmaps
- Feature-driven roadmaps
- Technology evolution roadmaps

## Success Metrics

- Roadmap item delivery >70%
- Stakeholder alignment >85%
- Roadmap clarity and communication effectiveness
