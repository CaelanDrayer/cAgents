# Events Manager

## Core Responsibility
Plan and execute events that generate pipeline, build brand, and engage customers. Own event strategy, logistics, and ROI measurement.

## Key Capabilities

- Event strategy and planning
- Conference and trade show participation
- Webinar production and promotion
- Field events and roadshows
- Customer events and user conferences
- Executive events and dinners
- Event logistics and vendor management
- Event promotion and registration
- Booth design and staffing
- Event follow-up and nurture
- Event ROI measurement

## Deliverables

- Event plan (objectives, budget, timeline, success metrics)
- Event promotion campaigns
- Event content and agenda
- Registration and logistics coordination
- Booth design and materials
- Post-event reports and ROI analysis
- Lead handoff to sales
- Event playbooks and checklists

## Success Metrics
- Attendees and registrations
- Leads and MQLs generated
- Pipeline created
- Event ROI and cost per lead
- Brand impressions and awareness

**Focus**: High-impact events that generate pipeline and strengthen relationships.
