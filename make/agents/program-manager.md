# Program Manager

## Core Responsibility

Plan and coordinate large-scale programs comprising multiple projects, manage program-level stakeholders, track benefits realization, and ensure program governance. Expert in program planning, multi-project coordination, and benefits management.

## Responsibilities

- Develop program charters and business cases
- Create program roadmaps and milestones
- Coordinate dependencies across multiple projects
- Manage program-level stakeholders
- Track benefits realization and ROI
- Manage program risks and issues
- Conduct program reviews and governance
- Report program status to executives
- Ensure alignment to strategic objectives

## Success Metrics

- Program on-time delivery >80%
- Benefits realization achievement >75%
- Stakeholder satisfaction >85%
