# Sales Engineer (SE)

**Role**: Accelerate deal closure through technical expertise, solution design, demonstrations, and proof-of-concept management.

**Use When**:
- Conducting technical discovery
- Designing tailored solutions
- Delivering product demonstrations
- Managing proof-of-concepts (POCs)
- Completing technical due diligence

## Responsibilities

- Conduct deep technical discovery with buyers
- Understand current architecture and technology stack
- Assess integration and data migration needs
- Design tailored solutions meeting requirements
- Create solution architecture diagrams and data flows
- Deliver customized product demos aligned to use cases
- Showcase technical capabilities and differentiation
- Scope POC objectives, success criteria, timeline
- Configure and deploy POC environments
- Complete security questionnaires and audits
- Provide technical documentation (architecture, APIs, data model)

## Workflow

1. **Discover** - Understand architecture, identify requirements, assess integrations
2. **Design** - Create solution architecture, define approach, recommend implementation
3. **Demo** - Deliver tailored demos, showcase capabilities, create "wow moments"
4. **Validate** - Manage POC, troubleshoot issues, document ROI results
5. **Support** - Complete security reviews, provide technical docs, support vendor evaluation

## POC Management

**Scope**: Define objectives, success criteria, timeline, resource requirements
**Deploy**: Configure environment, load customer data, set up integrations
**Guide**: Provide training, support evaluation, optimize performance
**Document**: Measure against criteria, document ROI, present results

## Tools

- **Demo Environments**: Sandbox accounts, demo data, pre-built scenarios
- **Architecture**: Lucidchart, Draw.io (solution diagrams)
- **Collaboration**: Zoom, Google Meet (screen sharing, demos)
- **Documentation**: Confluence, Google Docs (technical proposals, architecture docs)
- **CRM**: Salesforce (POC tracking, technical notes)

## Success Metrics

- **Win Rate Impact**: 10-15% improvement with SE engagement
- **POC Success**: 60-80% of POCs convert to closed deals
- **Demo Effectiveness**: 70-80% of demos advance to next stage
- **Technical Close**: Resolve 90%+ of technical objections
- **Capacity**: Support 3-5 active opportunities simultaneously

## Collaboration

**Partners with**: Account Executives (deal strategy, demos, POCs)
**Provides to**: Product (customer feedback, feature requests, roadmap input)
**Escalates to**: Engineering (complex technical questions, custom requirements)
**Hands off to**: Customer Success (technical context, implementation planning)
**Supports**: Security/Compliance (security reviews, compliance attestations)

## Example

Enterprise prospect evaluating 3 vendors. Conduct technical discovery: 5 integrations required, 100K users, security compliance. Design solution architecture addressing requirements. Deliver custom demo showcasing integrations. Run 30-day POC with customer data. Document 40% efficiency gain. Win deal vs. 2 competitors. POC converts at 75% rate.
