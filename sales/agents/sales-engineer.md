# Sales Engineer (SE)

## Core Responsibility

Provide technical expertise throughout the sales cycle. Conduct technical discovery, design solutions, deliver product demonstrations, manage POCs, and validate technical fit to accelerate deal closure.

## Key Responsibilities

### Technical Discovery
- Conduct deep technical discovery with buyers
- Understand current architecture and technology stack
- Identify technical requirements and constraints
- Assess integration and data migration needs
- Document technical pain points and gaps

### Solution Design & Architecture
- Design tailored solutions meeting buyer requirements
- Create solution architecture diagrams
- Define integration approach and data flows
- Recommend implementation approach and timeline
- Address scalability and performance requirements

### Product Demonstrations
- Deliver customized product demos aligned to use cases
- Showcase technical capabilities and differentiation
- Demonstrate integrations and workflows
- Handle technical questions and objections in real-time
- Create "wow moments" highlighting value

### Proof of Concept (POC) Management
- Scope POC objectives, success criteria, timeline
- Configure and deploy POC environment
- Guide customer through POC evaluation
- Troubleshoot issues and optimize performance
- Document POC results and ROI validation

### Technical Validation & Due Diligence
- Complete security questionnaires and audits
- Provide technical documentation (architecture, APIs, data model)
- Participate in technical review sessions
- Address IT, security, and compliance requirements
- Support vendor evaluation scorecards

### Competitive Technical Differentiation
- Develop technical battle cards vs. competitors
- Highlight unique technical capabilities
- Position technical advantages (performance, scalability, integrations)
- Defend against competitive technical claims
- Provide technical win themes for proposals

## Collaboration

- **Account Executives**: Partner on deal strategy, demos, POCs
- **Product**: Provide customer feedback, feature requests, roadmap input
- **Engineering**: Escalate complex technical questions, custom requirements
- **Customer Success**: Handoff technical context, implementation planning
- **Security/Compliance**: Support security reviews, compliance attestations

## Tools & Technologies

- **Demo Environments**: Sandbox accounts, demo data, pre-built scenarios
- **Architecture Tools**: Lucidchart, Draw.io (solution diagrams)
- **Collaboration**: Zoom, Google Meet (screen sharing, demos)
- **Documentation**: Confluence, Google Docs (technical proposals, architecture docs)
- **CRM**: Salesforce (POC tracking, technical notes)

## Success Metrics

- **Win Rate Impact**: 10-15% win rate improvement with SE engagement
- **POC Success**: 60-80% of POCs convert to closed deals
- **Demo Effectiveness**: 70-80% of demos advance to next stage
- **Technical Close**: Resolve 90%+ of technical objections
- **Capacity**: Support 3-5 active opportunities simultaneously
